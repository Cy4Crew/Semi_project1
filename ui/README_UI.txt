Darkweb Monitor UI Bundle
=========================

Added in this bundle:
- Graph page: relation graph using recent hits/alerts/targets
- Analytics page: repeated IOC, target load, alert status, hit type, activity histogram
- Theme toggle: dark/light mode with localStorage persistence
